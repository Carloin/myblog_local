##### 2.代码执行后，arr的值
```javascript
var arr=[]
for(var i=1;i<5;i++){
    (function(){
        setTimeout(function(){
            arr.push(i)
        },0)
    })()
}
arr.push(0)//0 5 5 5 5
```
