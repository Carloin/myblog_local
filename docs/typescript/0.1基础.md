## 1.快速上手
### 安装
```
> npm install -g typescript
```
###  构建第一个ts
* greeter.ts文件
```
function greeter(person) {
    return "Hello, " + person;
}

let user = "Jane User";

document.body.innerHTML = greeter(user);
```
* 运行
```
tsc greeter.ts
```
* 输出结果：为一个greeter.js文件，它包含了和输入文件中相同的JavsScript代码。
